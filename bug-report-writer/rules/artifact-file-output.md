# 产物文件输出规范

## 核心原则
**所有结果必须落地为文件，不能只在对话页面展示。**

## 输出路径规范
- Bug 单产物统一输出到：`outputs/reports/`
- 文件命名格式：`<module>-bug-<YYYYMMDD>.md`
  - 例：`order-payment-bug-20260323.md`
  - 例：`user-login-bug-20260323.md`
- 多个 Bug 可合并到同一文件，按 `## Bug-001`、`## Bug-002` 区分

## 强制要求
1. 每次执行结束必须给出完整文件路径
2. 文件路径使用相对路径（相对于 skill 根目录）
3. 不得只在对话中输出内容而不写文件
