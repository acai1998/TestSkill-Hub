## 一、测试策略
- 测试范围：用户手机号注册功能（输入手机号、获取验证码、设置密码、提交注册）
- 不测范围：第三方登录注册、企业账号注册、注册后的引导流程
- 关键风险：验证码发送频率限制、手机号重复注册、密码安全性校验
- 测试方法：功能测试、边界值测试、异常测试、安全测试、兼容性测试
- 测试环境与数据要求：测试环境、未注册手机号若干、已注册手机号1个、可接收短信的真实手机号

## 四、覆盖与追踪
| 需求点 | 覆盖说明 |
| --- | --- |
| 手机号格式校验 | 覆盖正常、少一位、多一位、含字母、已注册、空值共6条 |
| 验证码发送与校验 | 覆盖正确、错误、过期、60s内重发、60s后重发共5条 |
| 密码规则（6-20位） | 覆盖符合规则、5位、6位、20位、21位、空值共6条 |
| 安全性 | 覆盖密码加密传输、SQL注入、XSS注入共3条 |
| 兼容性 | 覆盖 iOS Safari、Android Chrome 共2条 |
| 异常场景 | 覆盖网络异常、已登录重访共2条 |
